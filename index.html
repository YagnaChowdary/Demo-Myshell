🐧Custom Linux Shell (C Project)

A fully functional Linux shell built from scratch using C, showcasing deep understanding of Operating System concepts, process control, and systems programming.

Project Overview

This custom shell replicates many core features of traditional Unix shells. It allows users to interact with the OS via a terminal interface, handling command parsing, I/O redirection, pipes, background processing, and signal management. Built as part of a B.Tech-level Operating Systems project.

Features

  • ✔️ Built-in commands like cd, exit, and jobs
  • ✔️ Background/foreground execution (&, fg, bg)
  • ✔️ Signal handling (Ctrl+C, Ctrl+Z)
  • ✔️ Command history
  • ✔️ Piping and I/O redirection
  • ✔️ Job control and process management
